# LibraryDRF

LibraryDRF is a Django REST Framework project that provides REST APIs for managing a collection of books. It supports full CRUD operations and includes a search feature to filter books based on a query.

## Overview

The application includes APIs for:

- Creating a new book  
- Listing all books  
- Retrieving a single book  
- Updating a book  
- Deleting a book  
- Searching books by title or author  

The project is intended for learning and demonstrating Django REST API development.

## Technologies Used

- Python  
- Django  
- Django REST Framework  
- SQLite (default)
